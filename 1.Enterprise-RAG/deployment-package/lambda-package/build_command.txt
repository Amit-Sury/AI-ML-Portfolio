1. Create tempoary folder and copy the requirements.txt into it.
2. Go to the temporary folder and pull Lambda base image from Docker Hub

docker pull amazon/aws-lambda-python:3.13

2. create a container from that image with something like:

docker run -it --rm -v /path/to/your/code:/var/task --entrypoint bash amazon/aws-lambda-python:3.13 

Here /path/to/your/code is the folder on your local machine where your code sits.
and /var/task is the working directory inside the container. e.g: 

docker run -it --rm -v "/$(pwd)":/var/task --entrypoint bash amazon/aws-lambda-python:3.13

After the above Docker command is executed, then we can check if the code is mapped correctly 
inside the container. After the container starts and show the command prompt, just run
ls /var/task. If the code files are listed there, that means the mounting 
worked perfectly. 

3.Once inside, run pip install commands to get the dependencies (! make sure to do it from within the
container)

# Create the proper Lambda layer directory structure
mkdir -p /var/task/python/lib/python3.13/site-packages

# Install packages from requirements.txt to the layer directory
pip install \
    --target /var/task/python/lib/python3.13/site-packages \
    --platform manylinux2014_x86_64 \
    --implementation cp \
    --python-version 3.13 \
    --only-binary=:all: \
    --upgrade \
    -r /var/task/requirements.txt

4. give "chmod 777 -R *" command

5. After that, exit from container. Use powershell and execute this

Compress-Archive -Path "python" -DestinationPath "lambda-layer.zip" -Force

Compress-Archive -Path "ingestion_service_lambda.py" -DestinationPath "lambda.zip" -Force

If performing manual upload then (Note: this is not required if deploying using deployment script): 

6. Upload both the files in s3 bucket. Create a layer using lambda-layer.zip 
(choose x86_64 as architecture and python 3.13 as environment)

7. Create lambda function use lambda.zip. Update runtime to give function name as 
"ingestion_service_lambda.lambda_handler". Add the created a lambda layer in step 6 to function
